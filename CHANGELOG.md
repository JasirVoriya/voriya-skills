# Changelog

This changelog tracks public releases for `voriya-skills`.

## v0.1.0 - March 6, 2026

This is the first tagged release for the repository. It establishes the
repository positioning, improves the onboarding path, and adds a repeatable
release workflow.

### Added

- A repository growth playbook in `docs/repo-growth-playbook.md`, including
  GitHub topics, a weekly release cadence, a release note template, and a
  distribution checklist.
- Trigger words and trigger scenarios in the Chinese and English skill guides
  for `website-capability-to-skill`.

### Improved

- The root landing pages in `README.md`, `README.zh-CN.md`, and
  `README.en.md` now lead with a clear value proposition, target audience, and
  a 10-second quick start.

### Notes

- This release starts versioned publishing for the repository.
